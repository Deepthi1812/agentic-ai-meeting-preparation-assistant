# Agentic AI Meeting Preparation Assistant

## Overview

This project is an AI-powered Meeting Preparation Assistant that helps users prepare for client meetings by gathering information from multiple sources.

## Features

- Retrieval-Augmented Generation (RAG) using FAISS
- Short-Term Memory for conversation context
- Long-Term Memory using JSON storage
- Agentic Workflow with tool usage
- Meeting Brief Generation

## Tools

1. Client Information Search
2. Meeting Notes Retrieval
3. Long-Term Memory Retrieval

## Tech Stack

- Python
- LangChain
- FAISS
- Ollama (Llama3)
- HuggingFace Embeddings

```bash
pip install -r requirements.txt
